# Municipal Grievance Redressal Management System

A Smart City solution for citizen grievance handling using Flask and MongoDB.

## Features
* User Role Management (Citizen, Officer, Admin)
* GIS-based Complaint Tracking
* Auto-assignment Engine
* Analytics Dashboard

## Deployment Guide

### 1. Prerequisites
* Python 3.9+
* MongoDB Atlas Account

### 2. Installation
```bash
git clone <repo-url>
cd municipal_grievance_system
pip install -r requirements.txt
